# 删除相关

删除文件有两种情况

# 删除文件

  • 删除文件

    • 点击删除后，会出现在 回收站

    • 只有清空了回收站，才算删除，硬盘才会释放空间

另：清空回收站，也并不代表完全已经从硬盘中删除，清空回收站释放空间后，在硬盘中的逻辑是暂时标记可用，但实际内容还可以通过数据恢复软件 恢复（有很多数据恢复软件，免费的也有，付费的也有，当然找专业人士恢复最稳妥，更有效的方式是自己检查到底要不要删除这个文件），但如果删除的数据已经被新数据替换，则可能不可再恢复

简单来说，清空回收站就像把书从图书馆的目录中删除，但书本身还在书架上。

# 文件、文件夹命名

有时候，安装一个软件，能安装，但是打开后报错，出现各种奇奇怪怪的错，如果某些软件在开发的时候没有考虑用户的文件路径是中文路径，安装的时候也会报错

这一点在开发、编程方面尤为重要，因为命名问题而导致程序不能运行，碰到的时候可是很折磨人的。

Q：为什么？

T：因为电脑只认识二进制 0 和 1，为什么英文可以？因为英文有 ascll 编码

建议使用英文：安装软件、编写代码

中英文随意：文档、音频（ppt、xlsx、txt、mp4、mp3……）

科普：ASCll 编码表，对应的英文（控制字符）在计算机中是 16 进制（ASCll 值），而 16 进制系统可以转换为二进制，但中文并没有在系统中存在这种表，这就是为什么推荐使用英文命名的原因

# 系统盘空间不足

电脑长时间使用后，磁盘空间不够了？为什么不够，有可能是你的软件只装在了一个磁盘分区中！

磁盘分区的作用就是为了方便存放数据，一般 C 盘为系统盘，是系统的启动盘，如果系统盘空间不够了，电脑就会出现卡顿，所以要养成习惯，不要把所有东西都存到 C 盘中，要学会管理自己的文件

例如：你有两个分区，C 盘就尽量不要存东西，把安装的软件都存到另一个盘

下载东西一般会用到浏览器，需要在浏览器设置下载文件的路径

  • 设置 ——> 下载 ——> 修改下载路径

QQ、微信的资源也可以进行迁移

需要尤其注意：桌面也属于 C 盘中的一部分，也可以自行修改路径（一般不建议，怕出现问题）

同样还是不建议将大文件放到桌面（如果经常用，图方便，也没关系，养成习惯后一般不会因为一个文件造成电脑卡顿）（养成这个习惯，你的电脑至少可以多用 3 年甚至更长）

# 安装包和软件的区别

Windows 的安装包一半分为三种

# .exe 结尾

  • .exe 结尾的安装包（之前讲的.exe 是可执行程序，当然也可以是安装程序）
  • 在下载之后，双击打开的是安装包，而不是直接运行的程序

# .msi 结尾

  • .msi 结尾的安装包，一般会安装服务（例如 MySQL），两者的差别并不大，只是.exe 结尾的，如果是单独的文件，有可能是安装包，也可能是单文件程序（很少，比如一个软件有广告，有人去除了广告，并且方便使用的时候才会出现）

自解压软件即为直接解压就可以打开用的（例如 V2rayN）

# 电量

电池：消耗品，充电次数越多，损耗越多，长时间处于满电状态也容易损坏电池，长时间不用也容易电池老化

优化：

  1. 保持在 70~90% 的电量（就是边充边用）
  2. 尽量减少充电次数，能减少电池损耗（就是边充边用）
